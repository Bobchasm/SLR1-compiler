# 编译原理大作业

**Deadline**: 2025年12月5日11:59PM (GMT+8)

## 一、目标

### 1. 背景知识介绍

- **C--语言**：C语言的子集，单文件（.sy后缀），去除了include/define/pointer/struct等复杂特性。
- **LLVM**：模块化、可重用的编译器和工具链集合，支持现代SSA编译策略。
- **LLVM IR**：LLVM中间代码，是源语言与目标架构之间的桥梁。

### 2. 大作业要求

编写编译器前端，包括：

1. **词法分析器**（基于自动机理论，可复用实验内容）
2. **语法分析器**（要求使用SLR算法）
3. **语义分析与中间代码生成**（补全代码，生成LLVM IR）

#### 必做任务：

- 词法分析器：识别单词符号，填写符号表
- 语法分析器：输入单词序列，输出推导/归约序列与语法树
- 中间代码生成：遍历语法树，调用中端代码生成LLVM IR

## 二、软件需求

### 1. 词法分析器

**输入**：C--源代码  
**输出**：单词符号序列 + 符号表

**单词类型**：

- **关键字（KW）**：int, void, return, const, main, float, if, else
- **运算符（OP）**：+, -, *, /, %, =, >, <, ==, <=, >=, !=, &&, ||
- **界符（SE）**：{ }, ( ), ; ,
- **标识符（IDN）**：字母/数字/下划线，不以数字开头
- **整数（INT）**：数字串
- **浮点数（FLOAT）**：带小数点的数字串

**实现语言**：C/C++/Java/Python（推荐C++）

### 2. 语法分析器

**输入**：单词符号序列  
**输出**：归约序列 + 语法树产生式序列

**支持语法功能**：

- 变量声明与初始化
- 常量声明
- 函数定义与参数管理
- 表达式与运算符
- 控制结构（if-else, return）

**要求**：

- 构造LR(0)项目集规范
- 计算FIRST/FOLLOW集
- 构造SLR分析表
- 输出归约过程与错误信息

**实现语言**：C/C++/Java/Python（推荐C++）

### 3. 中间代码生成

**输出格式**：`.ll`（人类可读的LLVM IR）

**任务**：补全Visitor函数，遍历语法树并调用中端代码生成IR

**实现语言**：C++

## 三、输出示例

### 1. 词法分析输出

```

a        <IDN,a>

...
```

```llvm

main_ENTRY:
    %op0 = load i32, i32* @a
    store i32 10, i32* @a
    ret i32 0
}
```
